Proyectos Recientes

Dead Run

Este videojuego está diseñado para sumergir a los jugadores en un mundo tenebroso y lleno de suspenso.
Se desarrolló en python y sin duda fue un gran reto pero una increíble experiencia llena de aprendizajes.

Hotelería

Esta es una página web intuitiva y hecha a la medida. Facilita las reservaciones y proporciona al recepcionista toda la información necesaria para ayudar al cliente a planificar su estancia, desde la descripción detallada de las habitaciones hasta las tarifas, actvidades disponibles y los datos de contacto.
Se desarrolló en Visual Studio Code, con Nextjs, Nestjs y React.

MegaTubos

MegaTubos es un software de inventario para una ferretería, cuenta con todo lo necesario para que la empresa almacene y venda sus productos.
Se desarrolló en Visual Studio con visual basic.
